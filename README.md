Blogify:

Blogify is a full-stack blogging website where users can create an account, write blogs, upload cover images, and comment on other blogs.

Features: 
User Signup and Login
JWT Authentication with Cookies
Create and View Blogs
Upload Cover Images using Multer
Comment on Blogs
User Profile Page
My Blogs Section
Display Blog and Comment Authors
Responsive UI using Bootstrap


Tech Stack
Node.js
Express.js
MongoDB
Mongoose
EJS
Bootstrap
JWT
Multer
